# Bank Term Deposit Prediction

## Опис

У цьому проєкті побудовано модель, яка передбачає, чи оформить клієнт банку 
строковий депозит.
Це задача бінарної класифікації.

## Дані

Датасет з Kaggle:
https://www.kaggle.com/datasets/sahistapatel96/bankadditionalfullcsv

* Дані про телефонні маркетингові кампанії банку
* Цільова змінна: `y` (yes / no)

## Що зроблено
* проведено EDA
* застосовано препроцесинг даних (кодування категоріальних ознак, масштабування)
* побудовано пайплайни за допомогою scikit-learn
* натреновано кілька моделей (Logistic Regression, kNN, Decision Tree, 
  XGBoost, LightGBM)
* оцінено якість моделей за метрикою ROC AUC

### Pipeline
Пайплайн включає:
* обробку пропущених значень
* кодування категоріальних ознак
* масштабування числових ознак
* навчання моделі

Це дозволяє виконувати всі кроки послідовно та уникати витоку даних.

## Структура проєкту

* `data/` — дані
* `notebooks/analysis.ipynb` — аналіз і експерименти
* `outputs/` — збережені результати 
* `src/eda_utils.py` — функції для EDA
* `src/pipelines.py` — пайплайни та моделі
* `requirements.txt` — залежності
